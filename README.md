## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/falconxi1611/employee-management.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start project
   ```sh
   npm start
   ```
